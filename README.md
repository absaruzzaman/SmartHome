# 🏠 SmartHome — Android Smart Home Control App

An Android application for monitoring and controlling IoT-enabled home devices  
such as lights and fans. Built as a Mobile App Development course project.

---

```
> cat about.txt
```

Three-person team project. I was responsible for the full UI layer —  
designing and implementing all application screens, navigation, and layouts.  
Teammates handled IoT communication and backend integration.

---

```
> ls screens/
```

```
auth/            → Login and authentication screen
dashboard/       → Main device control dashboard
device_control/  → On/Off controls for lights and fans
device_status/   → Real-time device status indicators
```
